# 커피 성격 퀴즈 - 요구사항 명세서

## 프로젝트 개요
사용자가 라이프스타일 질문에 답하면 커피 성격 타입과 추천 음료를 받는 웹 퀴즈

---

## 성격 타입 & 커피 페어링 (3개)

### 1. Cozy Classic (아늑한 클래식)
- **추천 커피**: Medium Roast Drip
- **태그라인**: "모든 잔에 편안함을"
- **특징**: 편안함, 안정감, 클래식한 취향

### 2. Health Nut (웰니스 추구자)
- **추천 커피**: Oat Milk Americano
- **태그라인**: "모든 한 모금에 웰니스를"
- **특징**: 건강 지향, 활동적, 균형 추구

### 3. Indulgent Treat (풍족한 디저트파)
- **추천 커피**: Mocha with Whip
- **태그라인**: "커피는 디저트다"
- **특징**: 즐거움 추구, 달콤함 선호, 삶을 즐김

---

## 결과 표시 로직

**방식**: 퍼센티지 표시 + 주요 추천 강조

### 예시 출력:
```
🎯 당신의 주요 타입: Cozy Classic (50%)
→ 추천 커피: Medium Roast Drip - "모든 잔에 편안함을"

당신의 다른 면모들:
• 33% Health Nut → Oat Milk Americano
• 17% Indulgent Treat → Mocha with Whip
```

### 계산 방식:
- 각 질문의 답변을 성격 타입에 매핑
- 전체 답변 중 각 타입이 선택된 비율 계산
- 가장 높은 퍼센티지를 주요 타입으로 표시
- 나머지 타입들도 퍼센티지와 함께 표시

## 시각적 스타일

### 디자인 컨셉
- **베이스**: 화이트 톤, 깔끔한 레이아웃 (미니멀)
- **인터랙션**: 마우스 오버/클릭 시 베이지 색으로 전환
- **느낌**: 심플하지만 따뜻하고 아늑한

### 컬러 팔레트
- 배경: 화이트/연한 회색 (#fafafa)
- 텍스트: 다크 그레이/블랙
- 인터랙션: 베이지 톤 (#e3d5ca, #d4c4b8)
- 강조: 따뜻한 갈색 (#6b4423, #9d7c5f)

### 타이포그래피
- 깔끔하고 읽기 편한 산세리프 폰트
- 충분한 여백과 라인 높이

### 레이아웃
- 중앙 정렬
- 넉넉한 패딩과 여백
- 모바일 반응형

## 퀴즈 질문 & 답변 매핑

### 질문 1: 주말 아침, 당신의 이상적인 시작은?
- **일찍 일어나서 조용히 책 읽기** → Cozy Classic
- **상쾌한 아침 운동으로 하루 시작** → Health Nut
- **늦잠 자고 브런치 즐기기** → Indulgent Treat

### 질문 2: 여행을 간다면 어떤 스타일을 선호하나요?
- **편안한 리조트에서 휴식** → Cozy Classic
- **하이킹이나 요가 리트릿** → Health Nut
- **미식 투어와 파인 다이닝** → Indulgent Treat

### 질문 3: 스트레스를 풀 때 당신의 방법은?
- **따뜻한 차 한 잔과 좋아하는 드라마** → Cozy Classic
- **명상이나 조깅으로 마음 정리** → Health Nut
- **맛있는 디저트와 와인** → Indulgent Treat

### 질문 4: 친구들이 당신을 뭐라고 설명할까요?
- **믿음직하고 편안한 사람** → Cozy Classic
- **에너지 넘치고 활동적인 사람** → Health Nut
- **삶을 즐길 줄 아는 사람** → Indulgent Treat

### 질문 5: 저녁 식사 후 당신의 선택은?
- **집에서 편하게 영화 보기** → Cozy Classic
- **산책하며 소화시키기** → Health Nut
- **디저트 카페에서 달콤한 마무리** → Indulgent Treat

## 기술 요구사항

### 프레임워크
- Next.js (React)
- JavaScript

### 기능
1. 순차적 질문 표시 (한 번에 하나씩)
2. 답변 선택 시 다음 질문으로 자동 이동
3. 진행 상태 표시 (예: "5개 중 3번째 질문")
4. 모든 질문 완료 후 결과 계산
5. 결과 페이지에 퍼센티지 표시
6. 반응형 디자인 (모바일 지원)

### 사용자 경험
- 부드러운 전환 애니메이션
- 명확한 피드백 (선택한 답변 하이라이트)
- 다시 시작 버튼
